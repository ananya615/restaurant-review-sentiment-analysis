# Restaurant Review Sentiment Analysis

This project analyzes restaurant reviews using Natural Language Processing (NLP) and Machine Learning (ML) techniques to classify sentiments as positive or negative. It was developed as part of the Data Science and Machine Learning (CS) course at Manipal University Jaipur (April 2025).

---

## Project Overview

Online restaurant reviews are a major factor in shaping customer opinions and influencing business decisions.  
This project automates the sentiment classification of restaurant reviews using machine learning algorithms and ensemble models, helping restaurant owners and analysts understand customer feedback more effectively.

---

## Features

- Text preprocessing (regex cleaning, stopword removal, stemming)
- Feature engineering using CountVectorizer and TF-IDF
- Sentiment scoring using VADER
- Additional feature extraction such as text length
- Implementation of multiple models:
  - Naive Bayes
  - Logistic Regression
  - Random Forest
  - Stacking Classifier
  - Voting Classifier
- Evaluation using accuracy, precision, recall, F1-score, and MCC
